# Bluesky MCP Server

MCP server for Bluesky social platform.

## Features

- Access Bluesky data via MCP
- Query posts, followers, and engagement metrics

## Configuration

| Variable | Required | Description |
|----------|----------|-------------|
| `BLUESKY_IDENTIFIER` | Yes | Bluesky handle (e.g., `user@bsky.social`) |
| `BLUESKY_PASSWORD` | Yes | App password or account password |
